How do I export a page set from TD Snap?
  1. In TD Snap, open Edit mode → Page Set → Import/Export.
  2. Choose Export to file and save the .sps/.spb file.
  3. Open that file here. When you're done, import the edited copy into a test TD Snap user first.

    Before you import — protect your real setup

    1. Keep your original export. The downloaded file is a separate .edited copy.
    2. Import into a test user first. In TD Snap, create a throwaway user and import there. If anything is wrong, your real setup is untouched.
    3. Exercise the page: open the parent page, tap the new link button, tap a few words, restart TD Snap, open it again.
    4. Only after that, import into your real user.
