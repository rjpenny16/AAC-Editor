Step 1

Connect to TD Snap

Changes go directly into the page set already open in TD Snap. There are no exports, duplicate files, or imports to manage.

Use the page set that is open now Leave TD Snap open and Windows unlocked while an edit runs.

Before you connect
  1. Open TD Snap and switch to the user and page set you want to edit.
  2. Open Topics, or open the page where you expect the new folder to belong. The app will suggest a location after you name it.
  3. Click Connect to TD Snap. It is okay if TD Snap is already in Edit mode.

Step 2

Plan a vocabulary edit

What do you want to do?

Choose the category first, then add words without creating another folder.

Also becomes the folder button label. A clear title helps the app recommend an existing page where it belongs.

Page style

Single words — each button speaks its label.

Communicative-function colors

Topic pages automatically start with blue questions, then orange comments, green positive responses, red negative responses, and purple personal messages. Move a button to another row to change its function, or click a button chip to override it.

Use Ollama instead (your own local model)

Choose this option if you already use Ollama or want to select a different local model.

  1. Install Ollama for Windows, macOS, or Linux, then open it.
  2. Open PowerShell or Terminal and run:
    ollama pull llama3.2
  3. Leave Ollama running. Return here and click Check connection.
