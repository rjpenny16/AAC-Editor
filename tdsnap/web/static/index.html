Step 1

Connect to TD Snap

Changes go directly into the page set already open in TD Snap. There are no exports, duplicate files, or imports to manage.

Connect to your TD Snap page set If TD Snap is closed, this opens it. Keep Windows unlocked while an edit runs.

Before you connect
  1. If TD Snap is already open, switch to the user and page set you want to edit.
  2. Open any page you want to preview or edit.
  3. Click Connect to TD Snap. The app opens TD Snap when needed, and it is okay if TD Snap is already in Edit mode.

Suggest words with AI optional · runs on this computer

AI suggestions are optional. The model runs on this computer; to keep suggestions accurate for specific topics (shows, games, characters), the app also looks up public reference facts online using the page title. That lookup can be turned off (set TDSNAP_WEB_GROUNDING=0), and suggestions still work offline without it.

Choose this option if you already use Ollama or want to select a different local model.

  1. Install Ollama for Windows, macOS, or Linux, then open it.
  2. Open PowerShell or Terminal and run:

  3. Leave Ollama running. Return here and click Check connection.
