Step 1

Connect to TD Snap

Open the page you want to change in TD Snap, then connect. The editor will read that page without moving any buttons.

Ready to connect?

Keep Windows unlocked while the editor is working.

Having trouble?
  1. Open TD Snap and choose the person and page set you want to edit.
  2. Open the page you want to change.
  3. Return here and select Connect to TD Snap.

Changes are applied directly. There are no exports to download or import.

Choose a task

What would you like to do?

The shorter path adds buttons to a page you already use.

Name the page

What should the new page be called?

Use a short, familiar name that will make sense in TD Snap.

This will also be the label on the button that opens the page.
