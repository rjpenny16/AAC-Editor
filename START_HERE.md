## 🎬 Your First 10 Minutes

### Minute 0-2: Installation
```bash
pip install -r requirements.txt
```

### Minute 2-3: Launch
```bash
# Windows
launch.bat

# Mac/Linux
./launch.sh
```

### Minute 3-6: Setup Coordinates
1. Open TD Snap
2. Go to "Setup Coordinates" tab
3. Record 5 button positions

### Minute 6-10: First Command
1. Go to "Command" tab
2. Type "Add colors"
3. Click "Process Command"
4. Watch it work!
