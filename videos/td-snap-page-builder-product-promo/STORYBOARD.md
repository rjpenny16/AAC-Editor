# TD Snap Page Builder Product Promo

38 seconds, 1920x1080, silent kinetic product demo based on HyperFrames' `product-promo` example.

| Time | Beat | On-screen message |
| --- | --- | --- |
| 0-3s | Brand pieces collide into a mark | Less editing. More communicating. |
| 3-12.5s | Existing TD Snap page set appears; cursor connects | Open the page set already in TD Snap. |
| 12.5-22s | Vocabulary chips flow into the live editor | Add the words you need - all at once. |
| 22-32s | Review checks resolve, then the update is approved | Review every change before it runs. |
| 32-38s | Brand lockup and benefit close | Private. Reviewable. Local. |
