# 🚀 QUICK START GUIDE

## Get Up and Running in 5 Minutes!

### Step 1: Install (2 minutes)
1. Install Python from https://www.python.org/downloads/ (if not already installed)
2. Open a terminal/command prompt in this folder
3. Run: `pip install -r requirements.txt`

### Step 2: Launch (10 seconds)
- **Windows**: Double-click `launch.bat`
- **Mac/Linux**: Double-click `launch.sh` or run `./launch.sh`

### Step 3: First-Time Setup (3 minutes)
1. Open TD Snap on your computer
2. Put TD Snap into **edit mode**
3. In the TD Snap AI Assistant window:
   - Go to "Setup Coordinates" tab
   - Click "Record" for "Add Category Button"
   - Within 3 seconds, hover your mouse over the "Add Category" button in TD Snap
   - Repeat for other buttons (Add Button, Label Field, etc.)

### Step 4: Start Using! (30 seconds)
1. Go back to the "Command" tab
2. Type: `Add restaurants category`
3. Click "Process Command"
4. Watch the magic happen! ✨

## ⚡ Pro Tips

1. **Start Small**: Try adding a category with just 5 items first to test your setup
2. **Watch It Work**: Don't minimize TD Snap while automation is running
3. **Adjust Speed**: If it's too fast, increase the delay in Settings tab
4. **Save Often**: Save your TD Snap work before running large automations

## ❓ Common First-Time Issues

**"Missing coordinates"**
→ You need to record button positions first (Step 3 above)

**Clicks in wrong place**
→ Make sure TD Snap is in the same position as when you recorded coordinates

**Too fast/slow**
→ Adjust "Delay between actions" in Settings tab
